# Python Learning Roadmap

## ✅ Topics You've Mastered
- Basic syntax (including statements)
- Output (text and numbers)
- Comments
- Variables (including multiple and global variables)
- Data types (strings, integers, floats, booleans)
- Numbers and casting (int(), float(), str())
- **Strings** (advanced string operations)
- **Booleans** (True/False, boolean logic)
- **Operators** (arithmetic, comparison, logical)
- **Lists** (creating, accessing, modifying)
- **Tuples** (immutable sequences)
- **Sets** (unique collections)
- **Dictionaries** (key-value pairs)
- **If-else statements** (conditionals)
- **Match statements** (pattern matching)
- **While loops** (conditional repetition)
- **For loops** (iteration)
- **Functions** (parameters/arguments, return/None, multiple returns, default parameters)
- **Range** (generating number sequences)
- **Arrays** (numpy arrays or list-based arrays)

---

## 📚 Next Topics to Learn

### 1. **Advanced Functions** (Optional but Useful)
- Keyword arguments (calling functions with `name=value`)
- Local vs global scope in functions
- `*args` and `**kwargs` (variable arguments)
- Lambda functions (optional)

### 2. **User Input** (Important for interactive programs)
- `input()` function
- Getting text input from user
- Converting input to different types
- Input validation basics
- Handling user errors

### 3. **List Methods & Operations** (Useful)
- List methods: `.append()`, `.remove()`, `.pop()`, `.sort()`
- List slicing: `my_list[1:4]`
- List comprehensions
- Nested lists

### 4. **Dictionary Methods** (Useful)
- Dictionary methods: `.keys()`, `.values()`, `.items()`
- Dictionary comprehensions
- Nested dictionaries
- Dictionary merging

### 5. **String Methods** (Useful)
- `.upper()`, `.lower()`, `.title()`
- `.strip()`, `.split()`, `.join()`
- `.replace()`, `.find()`, `.count()`
- String formatting: f-strings, `.format()`

### 6. **File I/O** (Useful for storing data)
- Reading from files: `open()`, `.read()`, `.readlines()`
- Writing to files: `.write()`, `.writelines()`
- Working with text files
- File paths and directories
- Context managers: `with` statement

### 7. **Error Handling** (Important)
- `try-except` blocks
- Handling specific exceptions
- `finally` clause
- Raising exceptions

### 8. **Modules and Packages** (Useful)
- Importing modules: `import`
- Standard library modules
- Creating your own modules
- Package structure

### 9. **Classes and Objects** (Advanced - Optional)
- Defining classes
- Creating objects
- Methods and attributes
- Inheritance basics

---

## 💡 Why These Topics?

These topics will help you build more sophisticated programs:
- **Complete Functions**: Organize and reuse code effectively
- **User Input**: Make interactive programs
- **List/Dictionary/String Methods**: Manipulate data efficiently
- **File I/O**: Save and load data persistently
- **Error Handling**: Make programs robust and user-friendly
- **Modules**: Organize larger projects
- **Classes**: Object-oriented programming (advanced)
