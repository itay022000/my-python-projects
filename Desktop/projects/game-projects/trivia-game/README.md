# Trivia Game Workspace

This repository will house multiple versions of the trivia game (and any future projects).
Each version lives in its own subdirectory so the code stays isolated and easy to evolve.

## Current Versions
- `versions/trivia_v1/` – the first playable version of the game (`trivia_v1.py`, `questions_v1.py`, docs)

## Running a Version
```
cd versions/trivia_v1
python3 trivia_v1.py
```

Add new versions by creating another subdirectory under `versions/` (e.g., `trivia_v2`) and
placing the updated files there.
